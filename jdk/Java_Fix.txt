Installation Instructions - Windows
--------------------------------------------------------------------------------
1. Extract and copy the JDK at the location
    C:\Program Files\Oracle\jdk-11.0.11+9

2. Add the JDK location in the PATH variable
    C:\Program Files\Oracle\jdk-11.0.11+9\bin

3. Create new variable JAVA_HOME with value
    C:\Program Files\Oracle\jdk-11.0.11+9      (beware -> no bin here)

4. Install preferred IDE
--------------------------------------------------------------------------------


Installation Instructions - Linux
--------------------------------------------------------------------------------
1. Extract and copy the linux JDK at the location
    /usr/local/bin/jdk-11.0.11+9

2. Make below changes to the ~/.bash_profile
    export JAVA_HOME="/usr/local/bin/jdk-11.0.11+9/"
    export PATH="${PATH}:/usr/local/bin/jdk-11.0.11+9/bin/"
--------------------------------------------------------------------------------
