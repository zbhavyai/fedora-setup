Installation Instructions
--------------------------------------------------------------------------------
1. Make sure JDK is properly installed with JAVA_HOME set-up

2. Extract the zip to somewhere. Example -
    C:\Program Files\Apache Maven\

3. Add the extracted "bin" directory to the PATH
    C:\Program Files\Apache Maven\bin

4. Open the file settings.xml using elevated privileges
    C:\Program Files\Apache Maven\conf\settings.xml

5. Change the localRepository path by adding below line
    <localRepository>D:\Softwares\Apache_Maven\Repository</localRepository>

6. For Maven 1, set up the environment variable MAVEN_HOME

7. For Maven 2, set up the environment variable M2_HOME

8. As of Maven 3.5.0, neither MAVEN_HOME nor M2_HOME is required

9. Confirm the installation by
    mvn -v
    mvn -version
